Site da premier League

É possivel abrir os links em uma nova guia do navegador usando a tag "Target" ="_blank" rel = "external" que é um comando novo que ajuda o navegador identificar que é um link externo que fica dessa maneira <a href="URL DO SITE" target="_blank" rel ="external"> nome do link </a>


Site do trivago

Por padrão os links vão abrir na mesma guia , mais caso deseja deixar isso explicito usasse o comando "_self" usando a tag "Target" ="_self" que fica dessa maneira <a href="URL DO SITE" target="_self"> nome do link </a>



Para dowload de arquivos usa o comando download="nome do arquivo/tipo do arquivo" <a href="URL DO SITE" dowload"ome do arquivo/tipo do arquivo"> nome do link para dowload </a>
